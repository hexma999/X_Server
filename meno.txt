Authorization
- http 인증 정보를 담기 위한 규칙
  
   Authorization:Basic WERWEFWVDSGEWREWREWRFWERF==

Bearer 
 - JWT 기반의 인증 규칙
 - 이 토근을 가지고 있는 사람은 인증된 사용자로 간주한다라는 의미 

   Authorization:Bearer WERWEFWVDSGEWREWREWRFWERF==  

브렌치
 같은 프로젝트 안에서 서로 다른 개발 라인을 만들고 각자 개발 후, 다시 협칠 수 있는 기능

 만드는 이유
 1. 많은 테스트를 해도 main 이 망가질 염려가 없음
 2. 기능별/db 별 작업 내용을 분리
 3. 여러명이 동시에 다른 기능을 개발할 때 충돌을 줄임
 4. 각자의 개발 완료 후 main 에 병합할 수 있음(실패 브렌치는 버리면 됨)  

 브렌치 목록 보기
 git branch
 
 git switch -c mysql-support
 또는
 git switch -c mysql-support(git checkout -b mysql-support)

기능별/브렌치명
git switch -c feature/mysql-support

이동
git switch main

삭제
git branch -d mysql-support
